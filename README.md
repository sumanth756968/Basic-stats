# 📊 Basics Statistics and Data Preprocessing

## 📌 Project Overview
This project demonstrates the fundamental concepts of descriptive statistics, data visualization, data preprocessing, standardization, and categorical variable encoding using Python. The analysis is performed in a Jupyter Notebook to understand the dataset before applying machine learning techniques.

---

## 🎯 Objectives
- Load and explore the dataset.
- Generate descriptive statistics.
- Separate numerical and categorical features.
- Calculate statistical measures such as Mean, Median, Mode, and Standard Deviation.
- Visualize data distributions using various plots.
- Standardize numerical features.
- Compare data before and after standardization.
- Perform One-Hot Encoding for categorical variables.
- Export the processed dataset.

---

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Project Workflow

1. Import required libraries.
2. Load the dataset.
3. Explore the dataset using:
   - `describe()`
   - `info()`
4. Separate numerical and categorical columns.
5. Compute:
   - Mean
   - Median
   - Mode
   - Standard Deviation
6. Visualize the data using:
   - Histograms
   - Boxplots
   - Bar Charts
7. Standardize numerical features using `StandardScaler`.
8. Compare data before and after standardization.
9. Apply One-Hot Encoding using `pd.get_dummies()`.
10. Save the transformed dataset as a CSV file.
